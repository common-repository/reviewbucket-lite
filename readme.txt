== Description ==
Reviewbucketlite is the plugin which allow you to display google reviews on your site from google using api. Also this plugin provide emoji reaction review feature. You could use emoji reaction review option on you post, page, product page or service page to get user reaction like facebook post reaction. This plugin helps you to increase your user confidence and trust.

== Installation ==

**The easy way.**

Go to your WordPress Dashboard. Navigate to Plugins > Add New and then search for "ReviewBucket Lite". Click on Install and then Activate the Plugin.

That's it, ReviewBucket Lite Plugin is now activated on your site!

**The hard way..**

Download "ReviewBucket Lite WordPress Plugin"  and then extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
